== Description ==
Protect posts in selected categories with a password. Users must enter the correct password to unlock and view the content beyond the "Read More" tag.

== Installation ==
1. Upload the `eba-content-protection` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `Settings -> Content Protection` to configure the plugin.
